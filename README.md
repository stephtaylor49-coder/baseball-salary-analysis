## Open the notebook in Colab (run it in your browser)
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/stephtaylor49-coder/baseball-salary-analysis/blob/main/BaseballSalary_ST.ipynb)

**How to Use This Repository**
**1. Open the Notebook**
  Click the “**Open in Colab**” button above.
  The notebook will open in your browser using Google Colab.
  Make sure you are logged into your Google account.

**2. Run the Notebook**
  In Colab, click Runtime at the top.
  Click **Run all**.
  If prompted, upload the required Excel file.
  The notebook will generate tables, charts, and output files.

**3. Make Changes to the Code**
If you edit the notebook, you must save your changes back to GitHub so everyone sees the latest version.
After making changes:
  Click **File**
  Click **Save a copy in GitHub**
  Choose the repository:
    **baseball-salary-analysis**
  Keep the file name the same:
    **BaseballSalary_ST.ipynb**
  Add a short description of what you changed
  Click **Save**
Now the updated version is stored in GitHub.

**4. Important Rules for Collaboration**
  Always open the notebook using the **Colab button in this repository**
  Always click **Run all** before making edits
  Always save changes back to GitHub when finished
  Do not upload output zip files or generated files to the repository

**5. If Something Looks Broken**
  Refresh the GitHub page
  Reopen the notebook using the Colab button
  Click **Runtime → Restart and run all**
